# Kalshi NBA Paper Trading - Frontend

React dashboard for the Kalshi NBA Paper Trading application.

## Setup

```bash
cd frontend
npm install
npm run dev
```

The frontend runs on http://localhost:3000 and proxies API requests to the backend at http://localhost:8000.

## Tech Stack

- React 18
- Vite (build tool)
- Tailwind CSS (styling)
- React Query (data fetching)
- React Router (navigation)
- Recharts (charts)
- Lucide React (icons)

## Pages

- **Games** - View and manage active games, place orders
- **Strategies** - Load and configure trading strategies
- **Trading** - Order history and positions
- **P&L** - Portfolio performance
- **Risk** - Risk management controls
